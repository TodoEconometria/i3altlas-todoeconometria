Stack Tecnologico

Big Data

  • Apache Spark 3.5 (cluster Docker)
  • PyArrow + Parquet
  • 50,000+ objetos astronomicos

Machine Learning

  • Random Forest, SVM, XGBoost
  • PCA + K-Means
  • Isolation Forest (anomalias)

Deep Learning

  • DNN Clasificador (Keras)
  • Autoencoder (anomalias)
  • 1D-CNN (espectral SDSS)

Datos Reales

  • JPL Small-Body Database (NASA)
  • JPL Horizons (efemerides)
  • SDSS Galaxy Colors (50K galaxias)
